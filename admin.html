使用说明

✅ 本系统现已支持 GitHub API 直写，配置 Token 后点击保存即可直接更新线上数据，无需手动操作。

🔑 首次使用请在「系统设置」中配置 GitHub Personal Access Token（需要 repo 权限）。

🎵 音乐作品支持 一键上传音频文件，在「音乐作品」页点击添加，选择文件即可自动上传到 GitHub。

🌐 保存后 GitHub Pages 会自动重新构建（约 1-2 分钟），刷新前台即可看到更新。
